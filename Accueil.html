Définition de la virtualisation

La virtualisation est une technologie qui permet de créer des versions virtuelles de ressources physiques (serveurs, systèmes d'exploitation, stockage, réseaux) pour optimiser leur utilisation et les gérer de manière indépendante.

Différence entre émulateur, simulateur et outil de virtualisation

Émulateur

Un émulateur recrée l’environnement matériel ou logiciel d’un système afin de permettre l’exécution d’applications comme si elles fonctionnaient sur ce système. Par exemple, un émulateur de console permet de jouer à des jeux conçus pour une console spécifique sur un autre appareil.

Simulateur

Un simulateur imite le comportement et les fonctionnalités d’un système réel sans reproduire son environnement exact. Il est souvent utilisé pour tester des applications dans des conditions spécifiques, comme la navigation GPS ou les vols virtuels.

Outil de virtualisation

Les outils de virtualisation permettent de créer plusieurs environnements virtuels sur un seul système physique. Ils sont utilisés pour exécuter différents systèmes d’exploitation ou applications de manière isolée, comme avec VMware ou VirtualBox.

Types de virtualisation pour les environnements VirtualBox, VmWare, Microsoft Hyper-V

Type 1

La virtualisation de type 1 (bare-metal) s'exécute directement sur le matériel physique, sans besoin d'un système d'exploitation hôte, offrant ainsi de meilleures performances et sécurité.

Exemple de VM : VirtualBox,VMware

Type 2

la virtualisation de type 2 fonctionne sur un système d'exploitation hôte, ce qui la rend plus simple à installer mais généralement moins performante.

Exemple de VM : Microsoft Hyper-V
